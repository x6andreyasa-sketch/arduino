Arduino Project

Alat Pendeteksi Banjir Sederhana Menggunakan Arduino Uno

Deskripsi Project

Alat pendeteksi banjir ini sangat simpel. karena menggunakan komponen dasar Arduino. Sistem bekerja dengan memanfaatkan water level sensor sebagai input, kemudian diproses oleh Arduino untuk menyalakan LED dan buzzer sebagai alarm.

Komponen yang Digunakan

  • Arduino Uno R3
  • USB Cable
  • Water Level Sensor
  • 3 LED
  • 3 Resistor
  • Kabel Jumper
  • Breadboard
  • Buzzer

Cara Kerja

Sensor water level membaca ketinggian air dan mengirimkan nilai analog ke Arduino. Jika air mulai terdeteksi:

  • LED 1 menyala (Level rendah)
  • LED 2 menyala (Level sedang)
  • LED 3 + Buzzer menyala (Level tinggi / bahaya)

Contoh Program Arduino

int lowerThreshold = 350;
int upperThreshold = 600;

// Pin sensor
#define sensorPower 7
#define sensorPin A0

// Variabel pembacaan sensor
int val = 0;

// Nada buzzer
int tones[] = {0, 277, 293, 311, 329, 349, 369, 392, 415, 440, 466, 493, 523, 554};

// Pin LED dan buzzer
int redLED = 2;
int yellowLED = 3;
int greenLED = 4;
int Buzzer = 5;

void setup() {
  pinMode(sensorPower, OUTPUT);
  digitalWrite(sensorPower, LOW);

  pinMode(redLED, OUTPUT);
  pinMode(yellowLED, OUTPUT);
  pinMode(greenLED, OUTPUT);
  pinMode(Buzzer, OUTPUT);

  Serial.begin(9600);
}

void loop() {
  val = readSensor();
  Serial.println(val);

  if (val <= lowerThreshold) {
    digitalWrite(redLED, HIGH);
    digitalWrite(yellowLED, LOW);
    digitalWrite(greenLED, LOW);
    noTone(Buzzer);
  }
  else if (val > lowerThreshold && val < upperThreshold) {
    digitalWrite(redLED, LOW);
    digitalWrite(yellowLED, HIGH);
    digitalWrite(greenLED, LOW);
    tone(Buzzer, tones[7]);
  }
  else {
    digitalWrite(redLED, LOW);
    digitalWrite(yellowLED, LOW);
    digitalWrite(greenLED, HIGH);
    tone(Buzzer, tones[10]);
  }

  delay(500);
}

// Fungsi membaca sensor air
int readSensor() {
  digitalWrite(sensorPower, HIGH);
  delay(10);
  val = analogRead(sensorPin);
  digitalWrite(sensorPower, LOW);
  return val;
}

Pembahasan Bagian Program

1. Penentuan Batas Level Air

int lowerThreshold = 350;
int upperThreshold = 600;

Dua variabel ini menentukan batas nilai sensor. - Di bawah 350 → Level rendah
- 350–600 → Level sedang
- Di atas 600 → Level tinggi / bahaya

2. Deklarasi Pin Sensor

#define sensorPower 7
#define sensorPin A0

Pin 7 digunakan untuk memberi daya ke sensor.
Pin A0 digunakan untuk membaca nilai analog dari sensor.

3. Variabel Pembacaan

int val = 0;
Variabel ini menyimpan nilai hasil pembacaan sensor air.

4. Array Nada Buzzer

int tones[] = {...};
Berisi daftar frekuensi nada (Hz).
tones[7] dan tones[10] dipakai untuk bunyi peringatan berbeda.

5. Deklarasi Pin Output

redLED = 2
yellowLED = 3
greenLED = 4
Buzzer = 5

Menentukan pin untuk LED dan buzzer.

6. Fungsi setup()

Digunakan untuk mengatur mode pin sebagai OUTPUT dan memulai komunikasi Serial Monitor dengan kecepatan 9600 baud.

7. Fungsi loop()

Program utama yang berjalan terus-menerus:

- Membaca nilai sensor
- Menampilkan nilai di Serial Monitor
- Menentukan kondisi LED dan buzzer berdasarkan level air
- Delay 500 ms agar pembacaan stabil

8. Fungsi readSensor()

Fungsi ini:
- Menghidupkan sensor
- Membaca nilai analog
- Mematikan sensor kembali
- Mengembalikan nilai pembacaan
